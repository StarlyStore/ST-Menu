### ⚡️ 메뉴  커맨드 ⚡️️
```yml

/메뉴 생성 <메뉴이름> : 메뉴를 생성합니다.

/메뉴 제거 <메뉴이름> : 메뉴를 제거합니다.

/메뉴 열기 <메뉴이름> : 메뉴를 엽니다.

/메뉴 목록 : 메뉴 목록을 확인합니다.

```
<hr>

### 📑 메뉴 펄미션 📑️
```yml

/메뉴 생성 : starly.menu.create

/메뉴 제거 : starly.menu.remove

/메뉴 열기 : starly.menu.open

/메뉴 목록 : starly.menu.lsit

```
